## 递归(recursive)
每个递归函数都有两部分：[参考文档3；有的算法书无此概念]
- 基线条件(base case)，函数不再调用自己的终结条件
- 递归条件(recursive case)，函数调用自己的条件

``` c++
void countdown(int i)
    cout<<i;
  if (i <= 1)  // 基线条件
    return ; 
  else         // 递归条件
    countdown(i-1);
```

## 分治算法(divide and conquer, D&C)
[参考文档3、4]
分治算法解决问题的步骤：
1. 找出基线条件，条件须尽可能简单；
2. 不断将问题分解(缩小规模)，直到符合基线条件。


快速排序、归并排序既是使用分治思想的算法。
